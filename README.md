# Proteger AI Frontend Assignment

This project is a frontend dashboard UI built using React.js as part of the Proteger AI assignment.

The implementation is based on the provided Figma design and focuses on clean code structure, proper component organization, and responsive UI behavior.

---

## Tech Stack
- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Chart libraries (if applicable)
- Dummy data for UI rendering

---

## Project Structure
The project follows a clean and modular folder structure with reusable components and clear separation between layouts, pages, and UI elements to ensure maintainability and readability.

---

## Responsiveness
The UI is designed to be responsive across desktop, tablet, and mobile screen sizes using standard responsive design practices.
